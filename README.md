📚 Book Library Management System

    A React-based Book Library Management System that allows users to browse, search, and manage books through a clean and user-friendly interface. The project demonstrates React Hooks, React Router, reusable components, API integration with Axios, and Express.js server configuration.

✨ Features:

* View all books
* Add new books
* Search books by title
* Book details page
* Responsive UI design
* React Hooks (useState, useEffect)
* React Router for navigation
* Express.js backend APIs
* Axios for API communication

## 🛠️ Technologies Used

Frontend:
- React.js
- JavaScript
- CSS3
- React Router DOM
- Axios

Backend:
- Node.js
- Express.js
- CORS

## 🚀 Installation

1. Install dependencies:

npm install

2. Start React Application:

npm start

3. Start Express Server:

npm run server
